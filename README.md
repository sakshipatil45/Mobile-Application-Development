## 🏛️ Academic Overview
This repository contains a structured collection of Android development projects completed as part of the **Mobile Application Development (MAD)** laboratory. Each project is designed to demonstrate proficiency in core Android concepts, from UI design to system-level integrations.

## 🎯 Learning Outcomes
By completing these experiments and evaluations, the following competencies were achieved:
- **UI/UX Design:** Proficiency in creating responsive and intuitive layouts using XML and Material Design.
- **Component Communication:** Mastering `Intents` for navigating between activities and passing data.
- **Event Handling:** Implementing robust listener mechanisms for user interactions.
- **Resource Management:** Efficient use of drawable, layout, and string resources.
* **Notification & Alerts:** Implementation of system-level feedback mechanisms.

## 📝 In-Semester Evaluation (ISE)

### **ISE 1: Student Information System**
- **Objective:** Develop a robust registration system for educational institutes.
- **Features:**
  - **Registration Form:** Detailed input validation for student records.
  - **List View:** Dynamically populated list of registered students.
  - **Data Handling:** Passing student objects between activities using Serializable/Parcelable.
- **Outcome:** Successfully demonstrated the ability to build a multi-screen application with complex data flow.
